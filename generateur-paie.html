Constantes Légales (Socle Universel)
Codées en dur — jamais modifiables par l'UI
PSS 2026 = 4 005 €/moisPlafond de la Sécurité Sociale. Mis à jour chaque année par décret. Sert de base pour toutes les tranches de cotisations.
CSG 6,80% + CRDS 0,50% = 9,20% totalTaux fixes universels. Base = 98,25% du brut (abattement de 1,75% pour frais professionnels).
Retraite SS plafonnée / déplafonnéeCes deux lignes ne peuvent JAMAIS être fusionnées. Bases différentes : T1 sur PSS, déplafonnée sur la totalité du brut.
Réduction Fillon (formule légale)Coefficient = (T/0,6) × (1,6×SMIC×12 / brut_annuel − 1). T = 0,3205 (TPE) ou 0,3245 (≥50 sal.). Plafonné à T, ≥ 0.
Famille 3,45% ou 5,25%Taux réduit si salaire ≤ 3,5×SMIC (≤ 6 306,30 €). Taux plein au-delà. Calcul automatique dans le moteur.
// LEGAL_2026 — immuable
const LEGAL = {

RET_DEP_S: 0.004,

// ...etc
};

Éléments Variables de Paie (EVP)
Modifiables via l'UI — propres à chaque entreprise/salarié
Mutuelle — montants en €/moisNi un taux ni une base standard. Défini par le contrat assureur et l'accord d'entreprise. Part patronale ≥ 50%.
Prévoyance — taux en %Variable selon la convention collective et le statut (cadre/non-cadre). Obligatoire pour les cadres (ACC 14/03/1947).
AT/MP — taux patronalNotifié par la CARSAT. Dépend du code risque APE et du taux d'accidentalité de l'entreprise. De 0,5% à +10%.
AGIRC-ARRCO T1/T2Taux contractuels. Les taux minimaux sont fixés, mais l'entreprise peut majorer par accord. T2 disparaît si brut ≤ PSS.
HS, Primes, Frais — conditionnelN'apparaissent que si actifs dans le mois. Le libellé est libre. Ces EVP modifient la base de cotisation.
// EVP — modifiable par l'UI
let evp = {
brut: 2800,
hs25: { on: false, nb: 0, label: "HS 25%" },
prime: { on: false, mt: 0, label: "Prime" },
mutuelle: { on: true, sal: 40, pat: 40 },
at_taux: 0.01, // VAR. entreprise
pas_taux: 0, // taux perso salarié
is_cadre: false,
};

Règles de conditions d'affichage
Logique displayRules — appliquée dans render()
// DISPLAY_RULES — chaque ligne a sa condition
const DISPLAY_RULES = {
// 🟢 TOUJOURS affiché (même à 0€)
maladie: () => true,
ret_pla: () => true, // ← jamais fusionner avec déplaf
ret_dep: () => true, // ← toujours ligne séparée
arrco_t1: () => true,
chomage: () => true,
csg_ded: () => true,
csg_nd: () => true,
pas: () => true, // ← PAS : toujours, même à 0%

// 🟡 CONDITIONNEL — masqué si non applicable
hs_25: (e) => e.hs25.on && e.hs25.nb > 0,
hs_50: (e) => e.hs50.on && e.hs50.nb > 0,
prime: (e) => e.prime.on && e.prime.mt > 0,
mutuelle: (e) => e.mutuelle.on,
prevoyance: (e) => e.prevoyance.on,
arrco_t2: (e) => e.brut_total > LEGAL.PSS, // ← uniquement si brut > PSS
transport: (e) => e.transport.on,
tr: (e) => e.tr.on,
apec: (e) => e.is_cadre,
fillon: (e) => calcFillon(e) > 0,
};
